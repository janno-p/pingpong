# Pingpong

Simple Pong game clone.

## Installation

Add this line to your application's Gemfile:

    gem 'pingpong'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install pingpong

## Usage

Start application by running:

    ruby bin/pingpong.rb
